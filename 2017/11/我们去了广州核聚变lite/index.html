我们去了广州核聚变Lite！

25 Nov 2017
Sylvia Wong

总结：核聚变Lite是轻松向的，场馆美术非常加分，整个会场的美术设计统一性极强，放眼望去每个角落都赏心悦目，主舞台的霓虹灯在全场都能看见，好看炸了。工作人员极其细心温柔，引导工作做得特别好，帅气小哥很多啊！徽章挑战难度不高，可能对高玩来说会过分简单了，但还是照顾了大部分人，墙上放了很多电视实时复制正在挑战的玩家的屏幕，排队的时候也不那么无聊可以看看，整体排队时间不算长，礼品很多，赞助商良心！就是在远一点的地方听不到舞台，要是排着队就有点遗憾，舞台互动挺有趣，玩家之间气氛也特别好，有种天下基佬是一家的亲切感。场子不大，但是大家都挺尽兴的。

以下是详细repo：

先介绍一下本人，手残，晕第一人称，爱看别人玩游戏，对各类游戏也就一知半解懂的不多，属于鶸玩家，最近才开始听机核电台。听说机核要来广州开核聚变兴奋到不行，赶紧召唤了个基友买了星期六的双人基佬票，后来转发微博又中了两张（第一次中奖！我爱华帝！），一行四人（电车之狼）星期六早上开开心心地搭有轨电车去会场。

// 一下电车就有机核插的flag做指引

入场要排一会儿队，门口有个Staff小哥跟我聊了会儿天。一口北京话听得我贼兴奋，但是新听众认不出脸也不好问人家是哪一位。我看那旗太好看了又拍了张照，无意中把旁边小哥拍进去了，后来才知道是哈迪！

// 机核巨好看的旗，还有知乎的赞助

之后发生了件巨好笑的事，我各路基友给我发消息说机核微博看见我了！之后我和你们哈英雄互粉了hhhhh巨开心

会场分为外场和内场，外场是吉考斯工业、其他周边购物区、核众食堂、华帝的互动区、休息区，休息区椅子有点少，到了下午大部分人都直接内场坐地上。

// 休息区前的提示

// 内场入口的灯牌，巨帅！

核聚变Lite设计了五个徽章，玩家必须去每个区域完成相关挑战才能获得徽章，有点像口袋妖怪打道馆的感觉。有些人有点懵比，在问集齐之后有没有特殊奖励，答案就是没有哈哈哈，给你留个纪念。

一进场人还不多，我们直奔育碧去打第一个徽章挑战，内容是刺客信条起源，清三波怪就挑战成功。

// 很有港味的霓虹灯

// 基友露出慈祥笑容

// 工作服帅炸了

// 穿了纪念T来的基友飞快拿下徽章

// 全是外星人，被壕气震慑！

因为我们去得早，尘埃4还没啥人排队，我没玩过赛车游戏，一上去简直一路乱开翻来翻去，Staff小哥看了看后面，对我说现在没人排队你再来一次吧，等一下人多就不可以了喔，呜呜呜对本废柴真是非常温柔了。结果第二次还是一路翻车，后来下午又排了两次逐渐掌握操作，但最后都是差一丢丢没能三分钟内跑完。很惭愧了！

// 临近中午渐渐人多，我们出去吃了个饭，早上晃悠一圈的战利品

// 下午回来发现人更多了，队伍这时候都好长，舞台前也聚集了很多人

// 街霸挑战，新听众一脸懵比

// 和高中同学面基

// 还见到了几年没见的朋友！巨开心！

外场华帝的人形娃娃机也挺有趣的，第一次吊钢索昂！还目睹了基友的朋友拿到了终极大奖NS！全场炸了！我夹出个移动电源哈哈哈这波不亏！

华帝的小零食，好看又好吃，不得不说活动在赞助这部分策划得太好了，每个赞助商都不讨人厌，大家都开开心心拿奖品。

听电台说归家异途制作人可能会来，结果好像没来。这游戏我之前Steam上买了，贼难，死活通不了。Staff是真心懂的，告诉我这最新版本跟PC上有哪里不同，我说我打了好几次都卡在电锯狂BOSS那儿了，小哥说走得那么远了呀差点就能通了。试玩五分钟领了个小徽章，小哥哥对我说你回去以后要把它打通喔！哥们看着有点眼熟，拼命回忆后好像是刚才尘埃4那位，好的！我会努力的！

// 待了大半天大家都累了，随便坐

// 你划我猜笑疯了，特别那星之卡比哈哈哈巨生动

最后的抽奖环节我们弄丢了一张券，血亏，获得NS的几率直线下降0.13%！

抽奖是全场的高潮，每当一个人“诶！“一声看手里的券，周围四面八方就会回头看过去哈哈哈贼有趣。我们电车之狼四人组啥也没中～到最后的大奖大家都自发给获奖者鼓掌，气氛真的太好了。不太舍得离场。我赶紧去跟Staff磨了一下有没有办法获得工作服，真的巨好看！娃娃机那几个小哥哥人超棒，悄悄告诉我还有新的，带我去找负责人问，还作势要把衣服脱给我hhh。很遗憾是不出售不交换的。

本人真是太残废了，除了华帝的章其他一个没拿到，基友可怜我帮我打了两个，不过我们四人都没集齐。另外今天overcooked的队实在太长了一直没玩到。

碰见非常多好玩的人，见识到很多硬核玩家，排队的时候听大家欢声笑语插科打诨也很有趣。机核说想做一个有点赛博朋克味道的展子，真的就做出来了，各方面还安排得那么周到。正如口号说的：“游戏唔觉难，最紧要好玩！“回票价那是当然值的，不仅拿了一堆奖品，还获得了非常美好的回忆。甚至小小点亮了我的未来。我前天才听的哈英雄那期节目，对那句“我总觉得我的人生不能就这么着吧！“特有感触。很佩服他能进入自己喜欢的行业做热爱的工作，我也不小了，大学快毕业了，也真心期望以后跟今天核聚变那群家伙一样活得认真，获得纯粹的快乐。夜深回忆起昨天的一切，各种场景还历历在目，感动得眼泪要掉下来。游戏就像是内心深处的一方净土，即使寸草不生，也一尘不染。一切像一场突如其来的梦，再见了！2017广州核聚变Lite！
